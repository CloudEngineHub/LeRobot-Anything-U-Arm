### Prerequisites

- Ubuntu 20.04
- ROS Noetic
- Python 3.9+

### Step-by-Step Setup

1. **Install Python Dependencies**

   ```bash
   pip install -r requirements.txt
   ```
2. **Build Catkin Workspace**

   ```bash
   catkin_make
   source devel/setup.bash
   ```
3. **Verify Installation**

   ```bash
   # Test if ROS can find the package
   rospack find uarm
   ```
